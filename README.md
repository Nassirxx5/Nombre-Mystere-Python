# 🎮 Jeu du Nombre Mystère (Projet Python)

Ce projet a été réalisé dans le cadre du module Python. Il s'agit d'un jeu interactif avec interface graphique développé en Python.

## 📝 Description du concept
Le **Nombre Mystère** est un jeu de devinette où l'utilisateur doit retrouver un nombre aléatoire entre 1 et 100. L'interface guide le joueur en indiquant si le nombre est plus grand ou plus petit, tout en comptant le nombre de tentatives.

## 🚀 Fonctionnalités principales
- **Génération aléatoire** d'un nombre cible.
- **Interface graphique** interactive avec Tkinter.
- **Indices visuels** (couleurs et texte) pour guider le joueur.
- **Gestion des erreurs** de saisie.
- **Pop-up de victoire** et bouton de réinitialisation.

## 🛠️ Technologies utilisées
- **Langage :** Python 3
- **Bibliothèque GUI :** Tkinter (standard)
- **Logique :** Module `random`
